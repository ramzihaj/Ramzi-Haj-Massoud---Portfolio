# CV Folder

## How to Add Your CV

1. **Save your CV** as a PDF file
2. **Rename it** to: `Ramzi_Haj_Massoud_CV.pdf`
3. **Place it in this folder**: `/public/cv/`

## File Requirements

- **Format**: PDF (Portable Document Format)
- **Name**: `Ramzi_Haj_Massoud_CV.pdf` (exact name)
- **Location**: `/public/cv/Ramzi_Haj_Massoud_CV.pdf`

## What Happens

Once you add the CV file:
- The "Download CV" button in the Hero section will work
- Visitors can download your CV with one click
- The file will be served directly from the public folder
